.. tip::

  Use wildcards to more effectively filter searches, i.e. testcases and filenames.

  +-----------+--------------------------------+
  | Pattern   | Meaning                        |
  +===========+================================+
  | \*        | match all                      |
  +-----------+--------------------------------+
  | ?         | match a single charecter       |
  +-----------+--------------------------------+
  | [seq]     | match any character in seq     |
  +-----------+--------------------------------+
  | [!seq]    | match all character not in seq |
  +-----------+--------------------------------+
